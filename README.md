# straights-gtk

![Screenshot](screenshot.png "Screenshot")

A modular implementation of the Straights card game using GTK+ 3.

## Installation

### Dependencies

 - libgtk-3-dev

### Building

```
git clone https://gitlab.com/jeffrey-xiao/straights-gtk
cd straights-gtk
make
./straights
```
